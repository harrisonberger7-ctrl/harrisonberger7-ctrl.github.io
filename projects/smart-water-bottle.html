Smart Water Bottle

An Arduino-powered water-dispensing prototype designed to improve accessibility for elderly and disabled users. The system monitors water level and temperature, displays live information on an LCD, and provides variable pump control through an analog joystick.

Project Overview

Designing an accessible automated drinking system

The Smart Water Bottle was developed to reduce the physical effort required to drink water. Instead of requiring the user to lift, tilt, or repeatedly open a conventional bottle, the prototype uses an electrically controlled pump to move water through flexible tubing.

An Elegoo UNO R3 reads the water-level sensor, waterproof temperature probe, and analog joystick. It then controls the pump and updates the LCD 1602 display with real-time operating data.

Primary Design Goals

  • Automate water dispensing
  • Monitor the remaining water level
  • Measure water temperature
  • Display live system information
  • Provide variable pump output
  • Prevent the pump from operating while dry

Hardware

Key system components

Elegoo UNO R3

Main microcontroller responsible for reading inputs, performing calculations, updating the display, and controlling the pump.

LCD 1602 Module

Displays pump output, water level, and measured water temperature in real time.

Analog Joystick

Provides variable user input that is converted into a pump-power percentage.

Water-Level Sensor

Measures the amount of water remaining and helps prevent the pump from operating while dry.

DS18B20 Temperature Probe

Waterproof digital probe used to measure the temperature of the stored water.

DC Water Pump

Moves water through the tubing with variable output determined by joystick position.

Switching Circuit

Transistor-based interface allows the Arduino to control the higher-current DC pump safely.

Prototype Hardware

Breadboards, jumper wires, resistors, battery connections, tubing, and enclosure materials complete the assembly.

Embedded Programming

Arduino control architecture

The Arduino sketch separates the major system functions into dedicated routines, making the code easier to test and organize.

WaterTemp()

Requests a reading from the DS18B20 sensor and converts the temperature from Celsius to Fahrenheit.

WaterLevel()

Reads the analog water-level signal and maps it to a percentage suitable for display and pump protection.

Joystick()

Converts joystick movement into a pump-control value ranging from zero to full output.

LCDScreen()

Updates the display with pump power, measured temperature, and remaining water level.

PumpON()

Converts the requested pump percentage into a PWM output for variable-speed dispensing.

PumpOFF()

Stops the pump whenever joystick input is absent or water level drops below the programmed threshold.

Mechanical Design

Prototype enclosure concept

The enclosure was designed around a central water reservoir with dedicated locations for the pump, temperature probe, water-level sensor, and supporting electronics.

Central Reservoir

The main cylindrical cavity contains the water while leaving space around the base for the pump and electrical hardware.

Sensor Integration

The water-level sensor is positioned vertically while the waterproof temperature sensor extends into the reservoir.

Electronics Cavity

A separate lower section provides room for the controller, wiring, power source, and pump circuitry.

Engineering Challenges

Integrating several systems into one prototype

  • Coordinating the joystick, pump, display, water-level sensor, and temperature probe within the main control loop
  • Developing conditional logic that prevented the pump from operating below the minimum water-level threshold
  • Calibrating an inconsistent analog water-level sensor
  • Mapping joystick position to a stable and understandable pump output percentage

Future Development

Potential design improvements

  • Replace temporary wiring with a soldered circuit or custom PCB
  • Develop a smaller, purpose-built enclosure
  • Use a longer and more reliable water-level sensor
  • Add a rechargeable internal battery
  • Improve cable management and waterproofing
  • Refine the design into a portable bottle-sized product
